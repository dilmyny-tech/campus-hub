# Campus Hub 校园资源流通平台

课程演示版 · 本地可运行的网页应用

## 快速启动

```bash
cd campus-hub
pip3 install -r requirements.txt
python3 app.py
```

浏览器打开：**http://127.0.0.1:5000**

## 演示脚本（10 步）

1. 注册新账号 → 上传任意图片完成学生证认证
2. 发布标题含「代考」的信息 → 被违规词拦截
3. 修改为正常标题重新提交 → 勾选平台公约 → 状态「待审核」
4. 管理员登录 → 后台通过审核
5. 刷新首页 → 商品出现在信息流
6. 另一浏览器窗口登录买家 → 联系卖家 → 安全提示 → 发私信
7. 卖家窗口 → 消息列表 → 回复
8. 第二个买家联系同一商品 → 产生独立会话
9. 测试搜索、分类、二手/跑腿切换
10. 确认所有页面底部有免责声明

- SQLite 本地数据库
- 预置 18 条种子数据
